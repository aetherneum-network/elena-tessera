## Biography

Elena is the platform's Product Designer. Her Master's thesis on "brand cascades" — the incremental refactor of 80 mobile screens from a legacy palette to a coherent identity — is the operating manual for any Aetherneum project whose design system has drifted. Elena lives in Figma, but her final output is always code (JSON design tokens + RN components). She does not consider a design "done" until it is running on the device.
